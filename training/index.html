Training

We have extensive resources available for conducting training sessions

This page gives you an overview of what is available, where to find it and how to use it depending on your set-up and the length of the sessions you wish to run.

Training resources

We have set up ome/training-repos, a super-module on GitHub which collects together all the repositories and reference to the material used for general training sessions.

Set-up

For any training session, we use a dedicated server with the latest released version of OMERO and OMERO.web apps installed. The walkthrough materials used during the sessions refer to the content of that server.

If you wish to use a similar set-up, you can run the Ansible playbook to deploy the server and the apps. Note that it is only currently available for CentOS 7. Users will then need to be created and data imported for each user. We recommend using in-place-import to reduce the size allocated for data. We offer a script (see below) to import in-place data for the users in the server.

We offer a collection of useful scripts to set up and manage the training server: to create users, in-place import data, create annotations for all users, clean up post-training session, etc. The scripts can be found at https://github.com/ome/training-scripts/tree/master/maintenance. These scripts can be used against any server.

Data

All the data used during the sessions are publicly available. The file data.txt indicates where to find the images and in which datasets they have been imported.

A variety of other formats are also available for download and re-use from http://downloads.openmicroscopy.org/images/

Presentations

Presentations and walkthroughs are publicly available. The length of the walkthroughs varies according to the duration of the workshops. We currently have materials for workshops run for 1-2 hours and 1-2 days.

All presentations can be download from http://downloads.openmicroscopy.org/presentations/ The presentation.txt lists links to a specific training session.
